## Key Performance Indicators (KPIs)

### **Primary Metrics**

- **ROAS**: Target 300% minimum (3:1 return)
- **CPA**: Target $30-50 (Customer Acquisition Cost)
- **Conversion Rate**: Target 2-4% (landing page)
- **CTR**: Target 3-5% (Search), 1-2% (Display/Social)

### **Secondary Metrics**

- Landing page bounce rate (target < 40%)
- Time on site (target > 2 minutes)
- Pages per session (target > 2.5)
- Return visitor rate

### **Optimization Triggers**

**Pause if**:
- ROAS < 150% for 7 days
- CPA > $75 for 7 days
- CTR < 1% for search ads

**Scale if**:
- ROAS > 400% for 7 days
- CPA < $30 for 7 days
- Conversion rate > 4%
